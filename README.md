# 3D NFT Project

A Foundry project for 3D NFT smart contracts.

## Getting Started

### Prerequisites

- [Foundry](https://book.getfoundry.sh/getting-started/installation)

### Installation

```bash
forge install
```

### Build

```bash
forge build
```

### Test

```bash
forge test
```

### Deploy

```bash
forge script script/Counter.s.sol:CounterScript --rpc-url <YOUR_RPC_URL> --broadcast --verify
```

## Project Structure

```
.
├── src/          # Smart contracts
├── script/       # Deployment scripts
├── test/         # Tests
└── lib/          # Dependencies
```
